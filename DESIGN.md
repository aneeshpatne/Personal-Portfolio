# Design System Document

## Theme: Fidelity Dark

This document outlines the core design principles and theme configuration for the "Fidelity Dark" design system.

### Color Palette

The color palette is built for a dark mode experience, emphasizing clarity and contrast.

* **Primary Color:** `#3B82F6` - A vibrant blue, used for primary actions, interactive elements, and key brand accents.
* **Secondary Color:** `#10B981` - A refreshing green, providing a complementary accent for secondary actions or supportive graphics.
* **Tertiary Color:** `#2A2A2C` - A deep, almost black, used for subtle highlights or specific UI components.
* **Neutral Color:** `#2A2A2C` - A dark, foundational gray, used for backgrounds, surfaces, and less prominent text.

### Typography

The typography system is designed for readability and modern aesthetics.

* **Headlines:** `spaceGrotesk` - Provides a distinctive, modern feel for titles and headings.
* **Body Text:** `manrope` - Chosen for its versatility and legibility in longer content blocks.
* **Labels:** `inter` - A highly functional and readable typeface ideal for UI elements, buttons, and form labels.

### Shape and Form

* **Roundedness:** `1` (Subtle roundedness) - Corners feature a subtle curve, providing a modern yet approachable aesthetic without feeling overly soft.

### Spacing

* **Spacing:** `2` (Normal) - Adheres to a standard spacing system, balancing visual density with adequate breathing room between elements for a comfortable user experience.
